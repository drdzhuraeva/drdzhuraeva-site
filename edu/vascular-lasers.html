Сосудистые лазеры — KTP, PDL, Nd:YAG

Купероз · розацеа · телеангиэктазии · по фототипам

Мишень — оксигемоглобин в сосуде. Длину волны выбирают по калибру и глубине сосуда; энергия нагревает и «запаивает» сосуд (фотокоагуляция).

Важно. Параметры — образовательный ориентир из опубликованных источников; требуют верификации по инструкции аппарата, подбираются по фототипу, калибру сосуда и тест-зоне.

PDL 595 нмСосочковая дермаРозацеа, эритема, винные пятна, гемангиомы
Nd:YAG 1064 нмГлубоко, крупные сосудыГлубокие/крупные сосуды, ретикулярные вены; безопаснее для тёмной кожи

Параметры и эндпоинты

  • Флюенс, длительность импульса (под калибр сосуда), размер пятна, охлаждение.
  • Эндпоинт: потемнение/спазм сосуда с сохранением тепла; для PDL допустима транзиторная пурпура (при коротком импульсе).
  • Длиннее импульс и Nd:YAG — безопаснее для смуглой кожи.

Показания

  • Купероз, розацеа, эритема, телеангиэктазии
  • Сосудистые мальформации (винные пятна, гемангиомы) — по показаниям
  • Часть пигментных задач (KTP)

Противопоказания

  • Активный загар (особенно для KTP); высокий фототип — осторожно
  • Приём фотосенсибилизаторов/изотретиноина; герпес/инфекция в зоне
  • Нарушения свёртываемости; беременность (относительно)
Список не исчерпывающий — сверяйтесь с инструкцией аппарата и собирайте анамнез.

Реабилитация

  • Эритема, лёгкий отёк; при PDL возможна пурпура несколько дней
  • Холод, мягкий уход, SPF 50; курс из нескольких процедур по показанию

Источники

  • Инструкции производителей (KTP/PDL/Nd:YAG-платформы)
  • Клинические рекомендации по сосудистым поражениям

Образовательный ориентир, требует верификации.
