# Embeddable-версии атласов для GetCourse

Самодостаточные интерактивные блоки (весь CSS и JS внутри, внешних
зависимостей нет, кроме Google Fonts). Предназначены для вставки в уроки
платформы GetCourse (или аналога), где доступ, оплата и лимит устройств
настраиваются средствами самой платформы.

## Файлы

- `botox-atlas.html` — ботулинотерапия (мышцы, сосуды, точки, векторы тяги).
- `lasers-atlas.html` — лазеры (срез кожи, глубины длин волн, фототипы).

## Способ 1 — вставка кода в урок (рекомендуется, полная защита)

Подходит, если тариф GetCourse **разрешает кастомный HTML/JavaScript** в уроке
(уточните в поддержке GetCourse: «можно ли вставлять произвольный HTML и
JavaScript в урок»).

1. Откройте нужный файл (`botox-atlas.html`) в текстовом редакторе.
2. Скопируйте всё между метками
   `<!-- ↓↓↓ КОПИРОВАТЬ ОТСЮДА ↓↓↓ -->` и `<!-- ↑↑↑ КОПИРОВАТЬ ДО СЮДА ↑↑↑ -->`
   (это блок `<style>` + `<div class="dz-edu">` + `<script>`).
3. В редакторе урока GetCourse вставьте этот код в блок «HTML-код» / «Исходный код».
4. Сохраните и проверьте в режиме ученика.

Контент в этом случае живёт внутри урока — публичной ссылки на него нет,
доступ полностью под контролем платформы.

## Способ 2 — встройка через iframe (если кастомный JS запрещён)

1. Файл должен лежать по доступному адресу (например, GitHub Pages:
   `https://drdzhuraeva.ru/edu/embed/botox-atlas.html`).
2. В урок вставьте:
   ```html
   <iframe src="https://drdzhuraeva.ru/edu/embed/botox-atlas.html"
           style="width:100%;height:1600px;border:0;" loading="lazy"></iframe>
   ```
3. Высоту (`height`) подберите под контент.

⚠ Минус способа 2: файл доступен по прямой ссылке всем, у кого она есть, —
защита слабее, чем при способе 1.

## Что настраивается на стороне GetCourse (не в коде)

- Приём оплаты и продажа доступа.
- Отдельный доступ на каждый курс/метод.
- **Лимит устройств / IP** и защита от копирования (антишеринг).
- Личные кабинеты учеников, выдача доступа после оплаты.

Эти пункты задаются в личном кабинете GetCourse и требуют вашего аккаунта
и реквизитов — их нельзя реализовать в статических файлах сайта.

## Обновление контента

Если меняете данные в основных страницах (`edu/botulinum-toxin.html`,
`edu/lasers.html`) — продублируйте правки в соответствующем embed-файле,
затем заново вставьте код в урок GetCourse.
