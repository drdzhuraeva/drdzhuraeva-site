# Образовательный портал для специалистов

Закрытый раздел сайта с клиническими материалами (аппараты, инъекционные
препараты, анатомия лица). Открывается из лендинга в отдельном окне.

## Структура

- `index.html` — хаб (каталог разделов).
- `botulinum-toxin.html` — пилотная страница (ботулинотерапия) с интерактивной
  анатомической SVG-моделью.
- `edu.css` — общие стили портала.
- `edu.js` — «мягкий» вход для специалистов (пароль + подтверждение).
- `anatomy-face.js` — интерактивность анатомической модели (слои + точки).

## Доступ / пароль

Вход — клиентский **барьер**, а не криптозащита: пароль виден в исходном коде
страницы. Подходит, чтобы отсечь случайных посетителей и зафиксировать
«я специалист», но не защищает контент технически.

- Сменить пароль: переменная `ACCESS_PASSWORD` в начале `edu.js`.
- Для **настоящей** закрытости нужен внешний сервис (платный мембершип,
  Netlify Identity / password protection, или бэкенд с авторизацией) —
  на чистом GitHub Pages это недостижимо.

## Анатомическая модель

Авторская SVG-схема (не копия атласов — оригинальная отрисовка). Слои:
мышцы, сосуды (артерии/вены), точки инъекций, векторы тяги мышц.
Данные точек задаются на странице через `window.ATLAS_POINTS`.

Чтобы переиспользовать модель на другой странице — скопируйте блок
`<section data-atlas>...</section>` и задайте свой `window.ATLAS_POINTS`.

## Статус контента

Все клинические параметры (дозы, глубины, настройки) — **образовательный
ориентир из опубликованных источников, требует верификации врачом**
(помечено бейджем «проверить»). Перед публикацией автор сверяет и утверждает.
